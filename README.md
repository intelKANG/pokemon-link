# 精灵连连看

复古连连看玩法的双端版本：网页版可以直接打开，安卓版已经打包成 APK。

## 试玩

- 网页版：打开 `index.html`，或发布到 GitHub Pages 后直接在线玩。
- 安卓版：安装根目录里的 `精灵连连看.apk`。

## 当前版本

- Web：正式版
- Android：2.2

## 玩法内容

- 经典连连看规则：连接路径最多转两次，消除相同小精灵。
- 支持暂停、提示、洗牌、重新开始、下一关。
- 关卡会逐步增加变化规则和难度，而不是只加快速度。
- 支持中文 / English，默认跟随系统语言，也可以在选项里切换。
- 网页端和安卓端共用同一套核心玩法。

## 关卡变化

游戏会循环使用这些经典变化规则，并随着关卡提高图标数量和时间压力：

1. 不变化
2. 向下移动
3. 向左移动
4. 向右移动
5. 向上移动
6. 上下分离
7. 左右分离
8. 向中聚拢

## 发布到 GitHub Pages

推荐把 GitHub Pages 的发布目录设置为：

- Branch：`main`
- Folder：`/web-release`

这样朋友打开 GitHub Pages 地址就能直接玩网页版。

## 重新打包 APK

如果需要重新生成安装包，运行：

```bash
./android-apk/build_apk.sh
```

脚本会把根目录的网页文件复制进 Android WebView 工程，并在根目录生成新的 `精灵连连看.apk`。
